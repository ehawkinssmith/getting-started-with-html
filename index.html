HTML

– basis for almost every webpage; glues everything together.

Files are hosted by servers and HTTP allows browsers to communicate with servers.

HTML is made up of:
text content
markup
references to other documents
links to other pages

HTML markup is made up of tags.

ex. < name >contents< /name >
This unit is referred to as an element.

Tag Examples

bold tag: < b >
em (emphasis) tag (italics): < i >

HTML Attributes

ex. < tag attr=”value” >contents< /tag >

Anchor: used for making links.
ex. the button

Image: used to include images
Ex. < img src=”url” alt=”text” > No closing tag required; called a void tag.

Whitespace

Use void tag < br > to create line breaks. Multiple break tags can be used to create multiple inline breaks.

Paragraph: < p >
Can be used to create "block" break. More info below.

Inline/Block

Inline ex.: < br >, < em >, < img >, < span >, < strong >
No invisible box around contents

Block ex.: < p >, < div >, < form >
Invisible box around contents; can have height and width.

HTML doc structure

Ex.
< !DOCTYPE HTML >
< html >
< head >
< title >Title< /title >
< /head >
< body >
< b >content< /b >
< /body >
< /html >
